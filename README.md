# 👗 VastraVillaVastraVilla 👗👔

Peer-to-Peer Clothing Rental Platform

<div align="center">

VastraVilla is a sustainable fashion platform that enables clothing sharing between individuals. The platform connects clothing owners with people looking for temporary clothing rentals, promoting sustainable fashion and reducing textile waste.

## 🚀 Getting Started

### Prerequisites

```bash
# Node.js & npm required
node --version
npm --version
```

### Installation

```bash
# Clone the repository
git clone https://github.com/ayushnautiyal-16/VastraVilla.git

# Navigate to project directory
cd VastraVilla

# Install dependencies (when available)
npm install

# Start development server
npm run dev
```
