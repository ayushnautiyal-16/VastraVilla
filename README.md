VastraVilla 👗👔
Peer-to-Peer Clothing Rental Platform

VastraVilla is a sustainable fashion platform that enables clothing sharing between individuals. The platform connects clothing owners with people looking for temporary clothing rentals, promoting sustainable fashion and reducing textile waste.

🎯 Project Overview
A dual-panel web application that facilitates peer-to-peer clothing rental:

Admin Panel: For clothing owners to list their second-hand clothes for rent
User Panel: For customers to browse and rent clothes based on their needs
✨ Key Features
Admin Features
List second-hand clothes for rent
Upload clothing images and details
Manage rental inventory
Track rental requests
Set rental prices and duration
User Features
Browse available clothing items
Filter clothes by type, size, occasion, and price
View detailed clothing information
Request rentals
Track rental history

🌱 Mission
Promoting sustainable fashion through:

Peer-to-peer clothing sharing
Reducing textile waste
Making fashion more accessible and affordable
Building a community of conscious consumers
